How to Play

A secret MLB player has been chosen. Guess players by name and you will get back a similarity score (0-1000) comparing your guess to the secret player. The higher the score, the more similar the careers. 1000 means you got it.

Scoring

You start with 101 points. Each guess costs 1 point. Each hint costs 20 points. Score floors at 0 but you can keep guessing. A correct answer scores whatever points remain, so a perfect game is 100.

Hints

Six hints are available, each with a different point cost based on how much they narrow the search:

  • INITIALS: 5 pts
  • BORN (birth year): 10 pts
  • COUNTRY: 10 pts
  • POS (primary position): 15 pts
  • WAR (career WAR): 15 pts
  • TEAM (primary team): 20 pts

Guess Badges

Each guess row shows five badges comparing your guess to the secret player, in the same order as the hints. Badge colors:

  • Green = exact match
  • Yellow = close but not exact
  • No color = far off

Badge logic for each field:

  • POS: green if same primary position, yellow if the guessed player played 10 or more games at the secret player's position
  • TEAM: green if same primary team, yellow if both teams are in the same 2025 MLB division
  • BORN: green if exact, yellow if within 5 years
  • WAR: green if within 2, yellow if within 10
  • COUNTRY: green if exact match only

Similarity Breakdown

Each guess shows which stat was most similar and which was least similar to the secret player.

Cross-Role Guesses

If the secret player is a pitcher and you guess a position player (or vice versa), you will get a score of 0. Use that as a signal to switch sides.

Similarity Score Formula

For position players, career stats compared include G, AB, R, H, 2B, 3B, HR, RBI, BB, SO, SB, BA, and SLG. For pitchers: W, L, SV, IP, K, and ERA. Based on the Baseball Reference similarity score methodology. All stats are as of Opening Day 2026.

Player Pool

The pool includes players who had meaningful MLB careers:

  • Position players: 500 or more career games
  • Starting pitchers: 100 or more career starts, with the majority of appearances as a starter
  • Relief pitchers: 300 or more career appearances, with the majority in relief

Year Range

The year range filter controls which players can be chosen as the secret player. It uses each player's actual first and last year in the majors. A player is included if their career overlaps with the selected range at all. Use the preset buttons or enter custom years.

Daily vs. Unlimited

Daily mode picks the same player for everyone on a given day. Unlimited mode picks a new random player each game.
